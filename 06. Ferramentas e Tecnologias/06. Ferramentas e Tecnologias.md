# 06. Ferramentas e Tecnologias

Esta seção consolida as ferramentas essenciais que compõem o ecossistema moderno de desenvolvimento, infraestrutura e produtividade.

---

## 💻 Desenvolvimento (IDEs e Editores)
As ferramentas onde o código ganha vida.
*   **Visual Studio Code (VS Code)**: O editor mais versátil e popular, com um ecossistema gigante de extensões.
*   **Visual Studio (IDE)**: Focado em desenvolvimento .NET robusto, enterprise e C++.
*   **Família JetBrains**: IDEs especializadas como **Rider** (.NET), **IntelliJ** (Java), **PyCharm** (Python) e **WebStorm** (JS/TS).

## 🐚 Terminal e Produtividade Shell
O terminal é a casa do engenheiro.
*   **Shells**: **Zsh** (com Oh My Zsh) ou **Fish** para uma experiência mais rica que o Bash padrão.
*   **Prompts**: **Starship** para um prompt rápido e informativo em qualquer shell.
*   **Emuladores**: **Windows Terminal**, **iTerm2** (macOS) ou **Alacritty** (Linux/Cross-platform).

## 🛠️ CLIs Essenciais (Operações)
Ferramentas de linha de comando fundamentais para o dia a dia.
*   **Orquestração**: `kubectl` (Kubernetes), `helm` (Package manager para K8s).
*   **Contêineres**: `docker` e `docker-compose`.
*   **Infraestrutura**: `terraform` e `ansible`.
*   **Cloud Providers**: `aws-cli`, `az` (Azure CLI) e `gcloud` (Google Cloud SDK).
*   **Versão**: `git` e `gh` (GitHub CLI).

## 📦 Gerenciadores de Pacotes e Build
*   **C# / .NET**: `dotnet CLI` e `NuGet`.
*   **JavaScript / TypeScript**: `npm`, `yarn` ou `pnpm`.
*   **Python**: `pip`, `poetry` ou `conda`.
*   **Sistemas**: `brew` (macOS/Linux), `apt` (Debian/Ubuntu) ou `choco`/`winget` (Windows).

## 🧠 Produtividade e Documentação
*   **Obsidian**: Gestão de conhecimento via Markdown e WikiLinks (a base desta vault).
*   **Excalidraw / Draw.io**: Criação de diagramas de arquitetura e fluxogramas.
*   **Postman / Insomnia**: Teste e exploração de APIs.
*   **DBeaver**: Gestão universal de bancos de dados SQL e NoSQL.
