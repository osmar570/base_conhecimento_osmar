# Frameworks de ML e IA

A escolha do framework certo depende do objetivo do projeto (pesquisa vs. produção) e da experiência da equipe.

## 🧱 Frameworks de Deep Learning

### 1. PyTorch (Meta)
Atualmente o favorito na academia e em laboratórios de pesquisa.
*   **Vantagens**: Gráfico computacional dinâmico, código mais "Pythonic", excelente ecossistema de bibliotecas (Hugging Face, PyTorch Lightning).
*   **Uso**: Prototipagem rápida e pesquisa de ponta.

### 2. TensorFlow (Google)
Focado em escalabilidade e deploy em produção.
*   **Vantagens**: Ecossistema robusto para produção (TFX, TensorFlow Serving), suporte para dispositivos móveis (TF Lite) e Web (TF.js).
*   **Uso**: Grandes infraestruturas corporativas e aplicações móveis.

### 3. Keras
Uma API de alto nível que pode rodar sobre o TensorFlow.
*   **Vantagens**: Extremamente fácil de usar, permite construir modelos complexos com poucas linhas de código.

## 📉 Machine Learning Clássico

### Scikit-learn
A biblioteca padrão para algoritmos de ML tradicionais.
*   **Capacidades**: Regressão, classificação, clustering, redução de dimensionalidade e pré-processamento de dados.
*   **Características**: Simples, eficiente e construída sobre NumPy e SciPy.

## 🎡 Ferramentas de Ecossistema
*   **Hugging Face Transformers**: A biblioteca definitiva para usar modelos pré-treinados (LLMs, Visão, Áudio).
*   **XGBoost / LightGBM**: Frameworks de Gradient Boosting extremamente performáticos para dados tabulares.
