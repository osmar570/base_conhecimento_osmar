# 🏗️ Estrutura de Dados

As estruturas de dados são formas organizadas de armazenar e gerenciar dados para que possam ser utilizados de maneira eficiente. Elas são a base para a construção de algoritmos robustos e sistemas de alta performance.

---

## 📂 Organização da Seção

Nesta seção, exploramos as principais estruturas de dados, suas características, complexidade (Big O) e casos de uso.

### 1. Estruturas Lineares
- **[[Arrays]]:** A forma mais básica de armazenamento contíguo.
- **[[Listas Ligadas]]:** Elementos dispersos na memória conectados por ponteiros.
- **[[Pilhas]]:** LIFO (Last-In, First-Out).
- **[[Filas]]:** FIFO (First-In, First-Out).

### 2. Estruturas Não-Lineares
- **[[Árvores]]:** Estruturas hierárquicas (Binárias, AVL, Red-Black).
- **[[Grafos]]:** Representação de conexões complexas entre nós.
- **[[Tabelas Hash]]:** Mapeamento de chave-valor para busca em tempo constante.

---

## ⏱️ Complexidade e Big O

Para cada estrutura, é fundamental entender o custo das operações principais:
- **Acesso**
- **Busca**
- **Inserção**
- **Remoção**
