# Qualidade de Código e Métricas

Manter a qualidade do código é essencial para a manutenibilidade e evolução sustentável de qualquer sistema de software.

## Principais Métricas
* **Complexidade Ciclomática**: Mede o número de caminhos linearmente independentes no código.
* **Cobertura de Código (Code Coverage)**: Percentual de código executado pelos testes.
* **Duplicação de Código**: Identificação de blocos repetidos (DRY - Don't Repeat Yourself).
* **Acoplamento e Coesão**: Grau de dependência entre módulos e foco de responsabilidade.

## Ferramentas de Análise
* **SonarQube**: Plataforma completa para inspeção contínua de qualidade.
* **ESLint / Prettier**: Linters e formatadores para ecossistemas JS/TS.
* **StyleCop / Roslyn Analyzers**: Para C#.
