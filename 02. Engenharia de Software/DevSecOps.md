# DevSecOps

DevSecOps é a integração de práticas de segurança em todas as fases do ciclo de vida de desenvolvimento de software (SDLC), do design ao deploy.

## Pilares do DevSecOps
1. **Security by Design**: Considerar segurança desde a concepção.
2. **Shift Left**: Mover os testes de segurança para o início do processo de desenvolvimento.
3. **Automação**: Integrar ferramentas de segurança no pipeline de CI/CD.

## Técnicas e Ferramentas
* **SAST (Static Application Security Testing)**: Análise do código fonte (ex: Snyk, SonarQube).
* **DAST (Dynamic Application Security Testing)**: Testes em tempo de execução (ex: OWASP ZAP).
* **SCA (Software Composition Analysis)**: Verificação de vulnerabilidades em dependências (ex: Dependabot).
* **Secret Management**: Proteção de chaves e senhas (ex: HashiCorp Vault).
