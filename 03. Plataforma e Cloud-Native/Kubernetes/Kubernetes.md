# O que é Kubernetes?

Kubernetes (também conhecido como **K8s**) é uma plataforma de código aberto (open-source) para a **automação do deploy, escalonamento e gerenciamento de aplicações em containers**.

## Principais Conceitos

- **Orquestração:** Gerencia o ciclo de vida dos containers, garantindo que eles estejam rodando conforme o esperado.
- **Escalabilidade:** Permite aumentar ou diminuir o número de instâncias de uma aplicação automaticamente com base na demanda.
- **Alta Disponibilidade:** Garante que a aplicação continue funcionando mesmo que algum nó (node) do cluster falhe.
- **Service Discovery:** Facilita a comunicação entre diferentes partes da aplicação sem a necessidade de configurações manuais complexas.

Originalmente desenvolvido pelo Google, o Kubernetes é hoje mantido pela **Cloud Native Computing Foundation (CNCF)**.
