# Trilha de Estudos: Desenvolvedor C# Júnior

O objetivo nesta fase é dominar a sintaxe da linguagem, os fundamentos da Programação Orientada a Objetos (POO) e ser capaz de entregar funcionalidades básicas seguindo o padrão do time.

---

## 🎯 Foco de Aprendizado

### 1. C# e .NET Fundamentals
- **Sintaxe básica:** Variáveis, operadores, estruturas de controle (if, switch, for, foreach).
- **Tipos de Dados:** Diferença entre Value Types e Reference Types.
- **Coleções:** Uso de `List<T>`, `Dictionary<K,V>` e introdução ao LINQ (Where, Select, FirstOrDefault).
- **POO Básica:** Classes, Objetos, Construtores, Propriedades e Métodos.

### 2. Desenvolvimento Web (ASP.NET Core)
- **Estrutura de API:** Controllers, Roteamento (Atributos) e Injeção de Dependência básica.
- **Middleware:** Entender o pipeline de requisição HTTP.
- **JSON:** Serialização e Deserialização com System.Text.Json.

### 3. Acesso a Dados
- **SQL Básico:** SELECT, INSERT, UPDATE, DELETE e JOINs simples.
- **Entity Framework Core:** Criar um DbContext e realizar operações de CRUD simples.

### 4. Ferramentas e Processos
- **IDE:** Atalhos essenciais do Visual Studio ou Rider.
- **Git:** Fluxo básico (Branch, Add, Commit, Push, Pull, Merge).
- **Depuração:** Uso de Breakpoints e inspeção de variáveis.
